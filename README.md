# ProceduralGrass
技術検証

自動生成による草の生成

パラメータにより、場所、回転、量などを指定して生成することができる

非ジオメトリシェーダーなので、環境依存はない


## Unity 2020.2 (URP + Shader graph)

- パラメータ
  - 風の速さ : 0.2
  - 風の強さ : 0.5
